# Docker layer structure

Thư mục này chứa các Docker assets tách theo vai trò:
- `frontend/` cho React/Vite
- `backend/` cho Spring Boot
- `nginx/` cho reverse proxy

Lưu ý: repo hiện vẫn dùng `frontend/Dockerfile` và `backend/educycle-java/Dockerfile` trong `docker-compose.yml` để không phá luồng chạy hiện tại.
